# telnyx-import-voice-ai

Skills for migrating voice AI assistants from other platforms into Telnyx.

Covers Vapi, Retell AI, and ElevenLabs — all using the `POST /v2/ai/assistants/import` endpoint.

## Included Skills

| Skill | Description |
|---|---|
| `telnyx-import-vapi` | Import Vapi assistants into Telnyx |
| `telnyx-import-retell` | Import Retell AI agents into Telnyx |
| `telnyx-import-elevenlabs` | Import ElevenLabs conversational AI agents into Telnyx |

Each skill covers all SDK languages (Python, JavaScript, Go, Java, Ruby, curl), selective import by ID, verification, and a post-import checklist.
