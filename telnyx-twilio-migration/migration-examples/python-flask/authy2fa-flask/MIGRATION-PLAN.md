## Unsupported Feature: OneTouch Push

**Critical**: This app uses Authy OneTouch push notifications for 2FA. Telnyx Verify does NOT support push notifications.

**Options**:
1. **Hybrid approach** (recommended): Keep Authy for OneTouch, migrate SMS OTP to Telnyx
2. **Full replacement**: Remove OneTouch, use only SMS verification via Telnyx
3. **Alternative**: Build custom push notification system with FCM/APNs

This migration will use option 1 — hybrid deployment.
