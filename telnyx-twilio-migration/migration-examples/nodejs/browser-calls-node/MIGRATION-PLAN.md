### Unsupported Product Detected

⚠️ **CRITICAL: This repository uses Twilio.Client (Twilio Voice WebRTC SDK)**

The Twilio Voice SDK (`Twilio.Device`) is a **browser-based WebRTC client**. Telnyx has a different approach:
- Telnyx WebRTC SDK (`@telnyx/webrtc`) uses direct SIP credential authentication
- No Access Token server-side generation required
- Architecture change: browser connects directly to Telnyx SIP infrastructure

**See:** `references/unsupported-products.md` for guidance on WebRTC client migrations.

**Key differences:**
- Twilio requires a backend token server (routes/token.js generates AccessTokens)
- Telnyx clients connect directly using SIP credentials or JWT tokens
- The /token/generate endpoint can be simplified or replaced
