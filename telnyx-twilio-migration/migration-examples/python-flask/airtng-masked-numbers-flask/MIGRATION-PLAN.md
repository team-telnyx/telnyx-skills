### Important Note on "Proxy" Pattern

This repository implements a **custom number masking solution** - NOT the official Twilio Proxy product. The code:
- Uses `available_phone_numbers` to search for numbers
- Uses `incoming_phone_numbers.create` to purchase numbers dynamically
- Routes calls/SMS between parties using these purchased numbers

This is a PRNG (do-it-yourself) masking solution that IS migratable to Telnyx using:
- Telnyx Number Pool API for number search/purchase
- Telnyx TeXML for call routing
- Telnyx Messaging API for SMS relay
